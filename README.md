# music-downloader

Download music from YouTube

## Requirements
1. Youtube-dl
2. BeautifulSoup
3. Ffmpeg

## Usage

```shell
$ ./downTheMusic.py
Enter songname / lyrics / artist ... or whatever
> we're not gonna take it
Searching ...
Found: We're not gonna take it - Twisted Sister
Download song (y/n)? y
Downloading ...
```
